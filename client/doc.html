Book Nook API Documentation

Accepted Methods

GET and HEAD

Query Parameters

No supported parameters.

Returns

A JSON array of all book objects in JSON database.

[{ author: string, country: string, language: string, link: string, pages: int, year: int, genres: [string], rating?: float }]

Example

Request: /getBooks

Result: [{ author: 'Hans Christian Andersen', country: 'Denmark', language: 'Danish', link: 'www....', pages: '784', title: 'Fairy tales', year: '1836', genres: ['Fairy Tales', 'Children's Literature'] }]

Accepted Methods

GET and HEAD

Query Parameters

title (required): A book's title to filter by.

Returns

A JSON array of all of the book objects in the JSON database that contain the string inputed by the user in their title.

[{ author: string, country: string, language: string, link: string, pages: int, year: int, genres: [string], rating?: float }]

Example

Request: /getBooksTitles?title=Fairy tales

Result: [{ author: 'Hans Christian Andersen', country: 'Denmark', language: 'Danish', link: 'www....', pages: '784', title: 'Fairy tales', year: '1836', genres: ['Fairy Tales', 'Children's Literature'] }]

Accepted Methods

GET and HEAD

Body Parameters

author (required): An author to filter by.

Returns

A JSON array of all of the book objects in the JSON database that contain the string inputed by the user in their author attribute.

[{ author: string, country: string, language: string, link: string, pages: int, year: int, genres: [string], rating?: float }]

Example

Request: /getBooksAuthor?author=Franz

Result: [{ author: 'Franz Kafka', country: 'Czechoslovakia', language: 'German', link: 'www....', pages: '488', title: 'Stories', year: '1924', genres: ["Folk Tales", "Classic Children's Literature"] }] [{ author: 'Franz Kafka', country: 'Czechoslovakia', language: 'German', link: 'www....', pages: '160', title: 'The Trial', year: '1925', genres: ['Modernism', 'Feminism'] }]

Accepted Method

POST

Supported Body Formats

application/json and application/x-www-form-urlencoded

Body Parameters

author (string, required): The author of the book.

country (string, required): The country of origin of the book.

language (string, required): The original language of the book.

link (string, required): A link to a website with more information about the book.

pages (number, required): The number of pages of the book.

year (number, required): The year the book was first published in.

genre (string, required): A book genre the book would belong to.

rating (number, required): Your personal rating for the book.

Returns

A JSON book object representing the new book.

[{ author: string, country: string, language: string, link: string, pages: int, year: int, genres: [string], rating?: float }]

Example

Request: /addBook

Body: [{ author: 'Franz Kafka', country: 'Czechoslovakia', language: 'German', link: 'www....', pages: '160', title: 'The Trial', year: '1925', genres: ['Modernism', 'Feminism'], rating: '5'}]

Accepted Method

POST

Supported Body Formats

application/json and application/x-www-form-urlencoded

Body Parameters

rating (number, required): Your personal rating for the book.

Returns

[{ author: string, country: string, language: string, link: string, pages: int, year: int, genres: [string], rating?: float }]

Example
